Attribute Selectors

This is a paragraph without title attribute.

p[title] selectsall paragraphs containing a title attribute.

p[title~="paragraph"] selects all paragraphs with a title attribute containing the word, paragraph.

p[title="#4 paragraph"] selects all paragraphs with a title attribute containing the exact text, #4 paragraph. Matches are case-sensitive and must match letter-for-letter including whilespace.
